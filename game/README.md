# Browser MMORPG

Браузерная 2D MMORPG игра с различными локациями, боевой системой, крафтом и квестами.

## Структура проекта

```
game/
├── index.html              # Главный HTML файл
├── css/
│   └── style.css          # Стили игры
├── js/
│   ├── main.js            # Основной контроллер игры
│   ├── data/
│   │   ├── constants.js   # Константы и конфигурация
│   │   └── gameData.js    # Данные игры (мобы, предметы, NPC, квесты)
│   └── modules/
│       ├── Player.js      # Класс игрока
│       ├── CombatSystem.js # Боевая система
│       ├── CraftingSystem.js # Система крафта
│       ├── QuestSystem.js # Система квестов
│       ├── UIManager.js   # Управление интерфейсом
│       ├── Renderer.js    # 2D рендерер на Canvas
│       └── DataManager.js # Управление данными
└── assets/                # Папка для ассетов (изображения, звуки)
```

## Локации

1. **Город** 🏰 - Стартовая позиция, безопасная зона
   - NPC: Старейшина (квесты), Торговец, Кузнец
   - Действия: Отдых, разговор с NPC, крафт

2. **Равнина** 🌾 - Первая боевая зона
   - Мобы: Слаймы, Зайцы, Курицы, Гоблины
   - Уровень мобов: 1-2

3. **Лес** 🌲 - Вторая боевая зона
   - Мобы: Волки, Медведи, Змеи, Кабаны
   - Уровень мобов: 3-5

4. **Пещера** ⛏️ - Третья зона с ресурсами
   - Мобы: Летучие мыши, Пауки, Скелеты, Големы
   - Ресурсы: Медная руда, Железная руда, Золотая руда, Кристаллы
   - Уровень мобов: 5-10

## Игровые системы

### Боевая система
- Пошаговый бой
- Атака, лечение, побег
- Награда: опыт, золото, лут

### Система крафта
- Рецепты из лута мобов и руды
- Зелья, оружие, броня

### Система квестов
- NPC выдают квесты
- Типы: убийство мобов, сбор предметов, добыча руды
- Награды: опыт, золото, предметы

### Экономика
- Золото за победу над мобами
- Продажа предметов
- Покупка у торговцев

## Запуск

Откройте `index.html` в браузере или используйте локальный сервер:

```bash
python3 -m http.server 8080
```

Затем откройте `http://localhost:8080`

## Управление

- **Навигация**: Кнопки локаций вверху
- **Действия**: Кнопки в панели действий
- **Инвентарь**: Использование предметов кликом
- **NPC**: Кнопка "Говорить с NPC" в городе
- **Крафт**: Панель крафта или через кузнеца

## Модульная архитектура

Игра построена по модульному принципу:
- Каждый игровой аспект выделен в отдельный класс
- Данные отделены от логики
- UI отделён от игровой логики
- Легко расширять новыми функциями
