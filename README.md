# 🎮 Browser MMORPG Game

Браузерная 2D MMORPG игра с разными локациями, квестами, крафтом и боевой системой.

## 🚀 Быстрый старт

### Требования
- Python 3.x (протестировано на 3.12+)

### Запуск игры

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd <repository-directory>
```

2. Запустите HTTP сервер:
```bash
python server.py [порт]
```

По умолчанию сервер запускается на порту **8000**.

3. Откройте в браузере:
```
http://localhost:8000
```

## 🎯 Особенности игры

### Локации
- **🏰 Город** - стартовая безопасная зона с NPC
- **🌾 Равнина** - первая боевая зона (слабые мобы: слаймы, зайцы, курицы, гоблины)
- **🌲 Лес** - вторая боевая зона (мобы посильнее: волки, медведи, змеи, кабаны)
- **⛏️ Пещера** - третья зона с мобами и рудами (медь, железо, золото, кристаллы)

### Системы
- ⚔️ Пошаговая боевая система
- 🎒 Инвентарь и экономика
- 🔨 Система крафта из лута и руды
- 📜 Квесты от NPC
- 👥 NPC: Старейшина (квесты), Торговец, Кузнец
- 🎨 2D визуализация на Canvas


## 🎮 Управление

- Перемещение между локациями через кнопки интерфейса
- Бой: выбор действий (Атака, Лечение, Побег)
- Крафт: создание предметов из ресурсов
- Квесты: получение и выполнение заданий от NPC

## 🛠️ Разработка

Сервер автоматически открывает браузер при запуске. Для остановки нажмите `Ctrl+C`.
